# Kostina Portfolio

Портфолио дизайнера, реализованное на современном React-стеке с использованием TypeScript и TailwindCSS.

## Технологический стек

### Основные технологии
- **React 18.3.1** — библиотека для построения пользовательских интерфейсов
- **TypeScript 5.5.3** — типизация JavaScript
- **Vite 5.4.2** — быстрый сборщик и dev-сервер
- **React Router 6.26.1** — клиентская маршрутизация

### Стилизация
- **TailwindCSS 3.4.10** — utility-first CSS-фреймворк
- **PostCSS 8.4.47** — обработка CSS
- **Autoprefixer 10.4.20** — автоматическое добавление вендорных префиксов
- **TailwindCSS Animate 1.0.7** — анимации для Tailwind

### Утилиты
- **Lucide React 0.445.0** — библиотека иконок
- **clsx 2.1.1** — условные классы
- **tailwind-merge 2.5.2** — слияние Tailwind-классов

## Установка

### Требования
- Node.js 18+ 
- npm 9+ или yarn 1.22+

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd Kostina_designer
```

2. Установите зависимости:
```bash
npm install
```

## Запуск проекта

### Режим разработки

Запуск dev-сервера с горячей заменой модулей (HMR):

```bash
npm run dev
```

Сервер будет доступен по адресу `http://localhost:5173`

Для доступа по сети:
```bash
npm run dev -- --host
```

### Сборка для продакшена

Создание оптимизированной сборки:

```bash
npm run build
```

Результат будет в папке `dist/`

### Предпросмотр сборки

Локальный предпросмотр продакшн-сборки:

```bash
npm run preview
```

## Конфигурация

### TypeScript
Конфигурация TypeScript находится в `config/tsconfig.json`. Проект использует строгую типизацию и современные настройки компилятора.

### TailwindCSS
Конфигурация TailwindCSS находится в `tailwind.config.ts`. Основные настройки:
- **Content** — пути к файлам для сканирования классов
- **Theme** — кастомизация цветовой палитры и стилей
- **Plugins** — подключение tailwindcss-animate

### Vite
Конфигурация Vite находится в `vite.config.ts` в корне проекта. Используется плагин `@vitejs/plugin-react` для поддержки React.

## Особенности проекта

- **Модульная архитектура** — компоненты разделены по папкам для удобства поддержки
- **TypeScript** — полная типизация для повышения надёжности кода
- **TailwindCSS** — быстрая разработка UI с готовыми утилитами
- **React Router** — клиентская маршрутизация для SPA
- **Оптимизированная сборка** — Vite обеспечивает быструю сборку и dev-сервер
